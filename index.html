@nrk/core-components

Getting started

core-components does not style any elements. It is purely a layer of functionality around some basic components, making sure accessibility is and extensibility is easy.

Use from npmjs.nrk.no

npm install @nrk/core-components --save
import components from '@nrk/core-components' // import all comonents...
import input from '@nrk/core-components/input' // ...or individual (recommended)

Use from static.nrk.no

<script defer src="https://static.nrk.no/core-components/X.X.X/core-components.min.js"></script>
